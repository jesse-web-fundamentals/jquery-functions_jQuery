Where to use: use .html() to operate on containers having html elements. use .text() to modify text of elements usually having separate open and closing tags Where not to use: .text() method cannot be used on form inputs or scripts. .val() for input or textarea elements. .html() for value of a script element. Picking up html content from .text() will convert the html tags into html entities. Difference: .text() can be used in both XML and HTML documents. .html() is only for html documents.
